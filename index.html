Logic

Let's start with True and False.

Logic Operators - AND

We can do arithmetic with logic

TRUE    AND    TRUE    =    TRUE
TRUE    AND    FALSE    =    FALSE
FALSE    AND    FALSE    =    FALSE

Logic Operators - OR

We can do arithmatic with logic

TRUE    OR    TRUE    =    TRUE
TRUE    OR    FALSE    =    TRUE
FALSE    OR    FALSE    =    FALSE

Data Types

TRUE and FALSE were so useful they become a regular type of data. What are some other types of data?

Data Types

In general there are 3 data types in any programming language.

Variables

With these types of data in mind, it is useful to be able to hold that data as something we can reference in different places. A popular concept in programming for this is a variable.

Variables

Creating a variable generally has these parts.
  1. Declare Variable Type (Optional in some languages)
  2. Give variable a name
  3. Give variable a value
