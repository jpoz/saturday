Logic

Logic Operators - AND

We can do arithmetic with logic

TRUE    AND    TRUE    =    TRUE
TRUE    AND    FALSE    =    FALSE
FALSE    AND    FALSE    =    FALSE

Logic Operators - OR

We can do arithmatic with logic

TRUE    OR    TRUE    =    TRUE
TRUE    OR    FALSE    =    TRUE
FALSE    OR    FALSE    =    FALSE
